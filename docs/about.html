What this blog is intended for...

This is a blog about Telia Operator Service Platform Opaali – specifically about its programming interface the Opaali API.

There will be example code snippets and blog posts explaining the code. There will also be a couple of full applications in source code form for you to use as almost drop-in replacements for some of the existing Content Gateway installations.

Who are we?

Although this blog and the accompanying source code is maintained by the Content Gateway to Opaali migration team, this is not part of the official Opaali product offering from Telia. During the migration period the migration team will update the blog and code irregularly. What happens after the migration has completed and the migration team will have disbanded – we don't know yet. The usual disclaimers apply ("Any opinions expressed here are those of the writer's and not..., you know"). The content of this GitHub repository comes AS IS with no guarantee from Telia whatsoever.
