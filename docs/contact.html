Don't call us...

This blog is a one way medium: there is no feedback channel, commenting is not enabled.

Your main contact for support in Opaali API related issues is still the {{ site.email_support }} mailbox. The quickest way to get an answer to a problem puzzling you is to drop a line to Opaali-tuki.

When needed, Opaali-tuki will forward your message to the migration team (that is us, who write this blog).

But before that, you should check out the official documentation at the Opaali Portal or the Opaali Portal Forum.
